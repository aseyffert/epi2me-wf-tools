EPI2ME Workflow Tools
=====================

A set of tools that facilitate genomic analyses via EPI2ME Labs
[workflows][epi2me-wfs] (e.g., their [human variation][wf-hv-ghub]
workflow) on the [CHPC][chpc-about] platform.

> [!WARNING]
> The software made available here is still under heavy development!
> Use at own risk!

[epi2me-wfs]: https://epi2me.nanoporetech.com/wfindex/
[wf-hv-ghub]: https://github.com/epi2me-labs/wf-human-variation
[chpc-about]: https://www.chpc.ac.za/about-the-chpc/
